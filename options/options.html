About This Extension

Fact-Check is a browser extension that uses AI to help you verify the accuracy of information you encounter online.

How It Works:
  • Select any text on a webpage to fact check it
  • The extension analyzes the text using OpenAI's advanced models
  • Results are presented with references and confidence levels
  • All processing happens through your own API keys for privacy
